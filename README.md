## ğŸš€ Quick Start

### Prerequisites

- Node.js >= 20
- Yarn v4.5.3 or later
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/mimir-labs/mimir-wallet.git
cd mimir-wallet
```

2. Install dependencies
```bash
yarn install
```

3. Start development server
```bash
yarn dev
```

Visit http://localhost:5173 to see the app.
